Details

Project ini merupakan final project untuk kelulusan dari Binar Academy Bootcamp pada kelas Full-Stack Web Development. Pengembangan dilakukan menggunakan metode SCRUM dan berkolaborasi dengan kelas Mobile Development. Tugas saya di antaranya:

  • Menjadi Scrum Master: mengelola backlog menggunakan Trello dan memastikan semua fitur delivered sebelum batas waktu pengumpulan.
  • Membuat API untuk digunakan FE Web dan Mobile
  • Membuat dokumentasi API menggunakan Swagger
  • Membuat Automated Unit Testing menggunakan JEST
  • Merapikan tampilan responsive

Penjelasan setiap fiturnya dapat dilihat DI SINI.
